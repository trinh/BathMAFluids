    2. A function with two branch points.

    Consider the function (1.2.4):
    \begin{equation*} f(z) = (z + 1)^{1/2} (z - 1)^{1/2}. \end{equation*}
    1. Choose the branch cut from \(z = 1\) in the positive real direction. Choose the branch cut from \(z = -1\) in the negative real direction. Write either \(z = r_1 \e^{\im\theta_1}\) or \(z = r_2 \e^{\im\theta_2}\) for \(\theta_1\) and \(\theta_2\) defined as relative angles from the two branch points. Show that: (i) when \(z = 1\) is encircled by a complete revolution, the function jumps in value by a factor of \(\e^{\im \pi}\text{;}\) (ii) that there is a similar jump in value when \(z = -1\) is encircled. Finally what happens if (iii) \(z = 0\) is encircled?
