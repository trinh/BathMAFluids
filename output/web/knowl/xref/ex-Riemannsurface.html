Exercise 1. Plotting a Riemann surface.

Select the branch cut of \(f(z) = z^{1/2}\) that runs along the positive real axis.

(a)

Consider a contour that starts from \(z = 1\text{,}\) then encircles the origin (anticlockwise) and returns to \(z = 1\text{.}\) What is the jump in the value of \(f(z)\) at the end of the contour as compared to the start?
ðŸ”—
Hint.
Let \(z = \e^{\im \theta}\) and consider \(\theta\) ranging from the initial value to a final value.
ðŸ”—
ðŸ”—
Solution.
Let \(z = \e^{\im \theta}\text{.}\) For \(\theta = 0\text{,}\) \(f(z) = r^{1/2}\) if we choose the positive branch of the square root (by convention). At the other side of the branch cut, \(\theta = 2\pi\) and \(f(z) = r^{1/2} \e^{\pi \im} = -r^{1/2}\text{.}\) Therefore there is a jump in value of \(-2r^{1/2}\text{.}\)
