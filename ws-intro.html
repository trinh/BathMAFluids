    1. Vector identities.

    Prove the following vector identities:
    1. \begin{equation*} \bnabla \cdot \left(\phi\mathbf{A}\right) = \phi (\bnabla \cdot \mathbf{A}) + \mathbf{A} \cdot (\bnabla \phi), \end{equation*}
      Answer.
      Using the Einstein summation convention,
      \begin{align*} \nabla \cdot (\phi \mathbf{A}) \amp = \pd{}{x_i}\left(\phi A_i\right)\\ \amp = \left(\pd{\phi}{x_i}\right)A_i + \phi \pd{A_i}{x_i}\\ \amp = \left(\bnabla\phi\right)\cdot\mathbf{A} + \phi\bnabla\cdot\mathbf{A}, \end{align*}
      where we have used the product rule in the second equality.
      🔗
      🔗
      🔗
    2. \begin{equation*} \bnabla \times (\phi \mathbf{A}) = \phi \left(\bnabla \times \mathbf{A}\right) + \left(\bnabla \phi\right) \times \mathbf{A} \end{equation*}
      Answer.
      We have,
      \begin{align*} \nabla \times (\phi \mathbf{A}) \amp = \epsilon_{ijk}\pd{}{x_j}\left(\phi A_k\right)\\ \amp = \left(\epsilon_{ijk}\pd{A_k}{x_j}\right)\phi+\epsilon_{ijk}\left(\pd{\phi}{x_j}\right)A_k\\ \amp = \left(\bnabla\times\mathbf{A}\right)\phi+\left(\bnabla\phi\right)\times\mathbf{A}. \end{align*}
      🔗
      🔗
      🔗
    3. \begin{equation*} \bnabla \cdot \left(\mathbf{A} \times \mathbf{B}\right) = \mathbf{B} \cdot \left(\bnabla \times \mathbf{A}\right) - \mathbf{A} \cdot \left(\bnabla \times \mathbf{B}\right) \end{equation*}
      Answer.
      We have,
      \begin{align*} \bnabla \cdot \left(\mathbf{A} \times \mathbf{B}\right) \amp = \pd{}{x_i}\left(\epsilon_{ijk}A_jB_k\right)\\ \amp = \epsilon_{ijk}\pd{A_j}{x_i}B_k+\epsilon_{ijk}A_j\pd{B_k}{x_i}\\ \amp = B_k\epsilon_{kij}\pd{A_j}{x_i}+A_j\epsilon_{jki}\pd{B_k}{x_i}\\ \amp = B_i\epsilon_{ijk}\pd{A_k}{x_j}+A_i\epsilon_{ikj}\pd{B_k}{x_j}\\ \amp = \mathbf{B}\cdot\left(\bnabla\times\mathbf{A}\right)-A_i\epsilon_{ijk}\pd{B_k}{x_j}\\ \amp = \mathbf{B}\cdot\left(\bnabla\times\mathbf{A}\right)-\mathbf{A}\cdot\left(\bnabla\times\mathbf{B}\right). \end{align*}
      🔗
      🔗
      🔗
    4. The following identity will be useful for proving the vorticity equation in Chapter 3 of this course:
      \begin{equation*} \bnabla\times\left(\bu\cdot\bnabla\bu\right) =\bu\cdot\bnabla\left(\bnabla\times\bu\right) +\left(\bnabla\cdot\bu\right)\left(\bnabla\times\bu\right) -\left(\left(\bnabla\times\bu\right)\cdot\bnabla\right)\bu \end{equation*}
      Answer.
      The \(i\)th component of the left-hand side equals
      \begin{align*} \amp\epsilon_{ijk}\pd{}{x_j}\left(u_l\pd{v_k}{x_l}\right)\\ =\amp\epsilon_{ijk}\pd{u_l}{x_j}\pd{v_k}{x_l}+\epsilon_{ijk}u_l\pd{^2v_k}{x_jx_l}. \end{align*}
      The second term of this expression equals the \(i\)th component of
      \begin{equation*} \bu\cdot\bnabla\left(\bnabla\times\bu\right). \end{equation*}
      🔗
      It therefore remains to prove that the first term
      \begin{equation*} \epsilon_{ijk}\pd{u_l}{x_j}\pd{u_k}{x_l} \end{equation*}
      equals the \(i\)th component of \((\bnabla\cdot\bu)(\bnabla\times\bu)-((\bnabla\times\bu)\cdot\bnabla)\bu\text{.}\) We note that if we can prove this for the first component then the proof for the second component follows by replacing index 1 by index 2, index 2 by index 3 and index 3 by index 1, and the proof of the third component by replacing index 1 by index 3, index 2 by index 1 and index 3 by index 2.
      🔗
      The first component of \((\bnabla\cdot\bu)(\bnabla\times\bu)-((\bnabla\times\bu)\cdot\bnabla)\bu\) equals
      \begin{align*} \amp\pd{u_l}{x_l}\epsilon_{1jk}\pd{u_k}{x_j}-\epsilon_{ljk}\pd{u_k}{x_j}\pd{u_1}{x_l}\\ =\amp\left(\pd{u_1}{x_1}+\pd{u_2}{x_2}+\pd{u_3}{x_3}\right)\left(\pd{u_3}{x_2}-\pd{u_2}{x_3}\right)\\ \amp -\pd{u_3}{x_2}\pd{u_1}{x_1}+\pd{u_2}{x_3}\pd{u_1}{x_1} -\pd{u_1}{x_3}\pd{u_1}{x_2}+\pd{u_3}{x_1}\pd{u_1}{x_2} -\pd{u_2}{x_1}\pd{u_1}{x_3}+\pd{u_1}{x_2}\pd{u_1}{x_3}\\ =\amp\left(\pd{u_2}{x_2}+\pd{u_3}{x_3}\right)\left(\pd{u_3}{x_2}-\pd{u_2}{x_3}\right)\\ \amp -\pd{u_1}{x_3}\pd{u_1}{x_2}+\pd{u_3}{x_1}\pd{u_1}{x_2} -\pd{u_2}{x_1}\pd{u_1}{x_3}+\pd{u_1}{x_2}\pd{u_1}{x_3}\\ =\amp\pd{u_2}{x_2}\pd{u_3}{x_2}+\pd{u_3}{x_3}\pd{u_3}{x_2} -\pd{u_2}{x_2}\pd{u_2}{x_3}-\pd{u_3}{x_3}\pd{u_2}{x_3}\\ \amp-\pd{u_1}{x_3}\pd{u_1}{x_2}+\pd{u_3}{x_1}\pd{u_1}{x_2} -\pd{u_2}{x_1}\pd{u_1}{x_3}+\pd{u_1}{x_2}\pd{u_1}{x_3}\\ =\amp\pd{u_2}{x_2}\pd{u_3}{x_2}+\pd{u_3}{x_3}\pd{u_3}{x_2} -\pd{u_2}{x_2}\pd{u_2}{x_3}-\pd{u_3}{x_3}\pd{u_2}{x_3}\\ \amp+\pd{u_3}{x_1}\pd{u_1}{x_2} -\pd{u_2}{x_1}\pd{u_1}{x_3} \end{align*}
      The \(i=1\) component of
      \begin{equation*} \epsilon_{ijk}\pd{u_l}{x_j}\pd{u_k}{x_l} \end{equation*}
      equals
      \begin{equation*} \pd{u_1}{x_2}\pd{u_3}{x_1}+\pd{u_2}{x_2}\pd{u_3}{x_2}+\pd{u_3}{x_2}\pd{u_3}{x_3} -\pd{u_1}{x_3}\pd{u_2}{x_1}-\pd{u_2}{x_3}\pd{u_2}{x_2}-\pd{u_3}{x_3}\pd{u_2}{x_3}. \end{equation*}
      Inspecting these expressions, we find they are equal. QED.
      🔗
      🔗

    2. Divergence theorem.

    State the divergence theorem. Use it to show that, for a vector-valued function \(\mathbf{B}(\mathbf{x})\text{,}\)
    \begin{equation*} \int_V \bnabla \times \mathbf{B} \, dV = \int_{\partial V} \hat{\mathbf{n}} \times \mathbf{B} \, dS, \end{equation*}
    where \(\hat{\mathbf{n}}\) is the outward-pointing unit normal to the surface \(\partial V\text{.}\)
    🔗
    Answer.
    The divergence theorem states that, for a vector field \(\mathbf{A}(\mathbf{x})\text{,}\)
    \begin{equation*} \int_V \bnabla \cdot \mathbf{A} \, dV = \int_{\partial V} \hat{\mathbf{n}} \cdot \mathbf{A} \, dS. \end{equation*}
    🔗
    To prove the required result, we use the Levi--Civita symbol \(\epsilon_{ijk}\) to write the \(i\)th component of the left-hand side as
    \begin{align*} \int_V (\bnabla \times \mathbf{B})_i \, dV \amp = \int_V \epsilon_{ijk}\pd{B_k}{x_j} \, dV\\ \amp = \epsilon_{ijk}\int_V \pd{B_k}{x_j} \, dV \end{align*}
    Setting \(\mathbf{A}=B_k\mathbf{e}_j\text{:}\)
    \begin{equation*} \bnabla\cdot\mathbf{A}=\pd{B_k}{x_j},\quad \hat{\mathbf{n}}\cdot\mathbf{A}=\hat{n}_jB_k. \end{equation*}
    and the divergence theorem implies
    \begin{equation*} \int_V \pd{B_k}{x_j} \, dV = \int_{\partial V} \hat{n}_j B_k \, dS. \end{equation*}
    Hence,
    \begin{align*} \int_V (\bnabla \times \mathbf{B})_i \, dV \amp = \epsilon_{ijk}\int_{\partial V} \hat{n}_j B_k \, dS\\ \amp= \int_{\partial V} (\hat{\mathbf{n}} \times \mathbf{B})_i \, dS. \end{align*}
    QED.

    3. Stokes’ theorem.

    State Stokes’ theorem. Use it to evaluate the line integral \(\oint z\,dx+x\,dy+y\,dz\) around the triangle with vertices \((3,0,0)\text{,}\) \((0,0,2)\) and \((0,6,0)\text{.}\)
    🔗
    Answer.
    Stokes’ theorem states that, for a vector field \(\mathbf{A}(\mathbf{x})\) defined on a surface \(S\text{,}\)
    \begin{equation*} \int_S \left(\bnabla \times \mathbf{A}\right) \cdot \hat{\mathbf{n}} \, dS = \oint_{\partial S} \mathbf{A} \cdot d\mathbf{x}. \end{equation*}
    where \(\hat{\mathbf{n}}\) is the unit normal to the surface \(S\) and \(\partial S\) is the boundary curve of \(S\text{.}\)
    🔗
    Setting \(\mathbf{A}=(z,x,y)\text{,}\) and noting that \(\bnabla \times \mathbf{A} = (1,1,1)\text{,}\) we have that, for any surface \(S\) spanning the given triangle,
    \begin{equation*} \int_S (1,1,1)\cdot\hat{\mathbf{n}} \, dS =\oint_{\partial S} (z,x,y)\cdot d\mathbf{x} =\oint_{\partial S} z\,dx+x\,dy+y\,dz. \end{equation*}
    Hence, the required line integral equals the flux of the vector \((1,1,1)\) through the surface \(S\text{.}\) We choose \(S\) to be the triangle itself.
    🔗
    The two edges of the triangle that start from the vertex \((3,0,0)\) are given by the vectors \(\mathbf{a}=(-3,0,2)\) and \(\mathbf{b}=(-3,6,0)\text{.}\) The cross product of these two vectors is
    \begin{equation*} (-3,0,2)\times(-3,6,0)=(-12,-6,-18). \end{equation*}
    This vector is orthogonal to \(S\) and its magnitude equals the area of the parallelogram with these vectors as two of its edges. The triangle has area half that of the parallelogram \(\frac12\sqrt{(-12)^2+(-6)^2+(-18)^2}=3\sqrt{2^2+1^2+3^2}=3\sqrt{14}\text{,}\) and its unit normal is \(\hat{\mathbf{n}}=(-2,-1,-3)/\sqrt{14}\) (note that we can check the direction of the normal using the right-hand rule). Hence
    \begin{equation*} \oint_{\partial S} z\,dx+x\,dy+y\,dz =\int_S (1,1,1)\cdot\hat{\mathbf{n}} \, dS =3\sqrt{14}(1,1,1)\cdot\frac{(-2,-1,-3)}{\sqrt{14}} =-18. \end{equation*}
    🔗

    4. Coordinate systems.

    1. A vector field is given in Cartesian coordinates by \(\mathbf{A}=(x,y,0)\text{.}\) Convert to cylindrical and spherical polar coordinates. What do you notice?
      Answer.
      For cylindrical polars, we have \(x=r\cos\theta\text{,}\) \(y=r\sin\theta\) and \(z=z\text{.}\) We may also write
      \begin{equation*} \hat{\mathbf{r}}=\cos\theta\,\hat{\mathbf{x}}+\sin\theta\,\hat{\mathbf{y}}. \end{equation*}
      Hence,
      \begin{align*} \mathbf{A} \amp = x\,\hat{\mathbf{x}} + y\,\hat{\mathbf{y}} + 0\,\hat{\mathbf{z}}\\ \amp = r\cos\theta\,\hat{\mathbf{x}}+r\sin\theta\,\hat{\mathbf{y}}\\ \amp=r\hat{\mathbf{r}}. \end{align*}
      🔗
      For spherical polars, we have \(x=r\sin\theta\cos\phi\text{,}\) \(y=r\sin\theta\sin\phi\) and \(z=r\cos\theta\text{.}\) We may also write
      \begin{equation*} \hat{\mathbf{r}}=\sin\theta\cos\phi\,\hat{\mathbf{x}}+\sin\theta\sin\phi\,\hat{\mathbf{y}}+\cos\theta\,\hat{\mathbf{z}} \end{equation*}
      and
      \begin{equation*} \hat{\boldsymbol{\theta}}=\cos\theta\cos\phi\,\hat{\mathbf{x}}+\cos\theta\sin\phi\,\hat{\mathbf{y}}-\sin\theta\,\hat{\mathbf{z}}. \end{equation*}
      Hence,
      \begin{align*} \mathbf{A} \amp = x\,\hat{\mathbf{x}} + y\,\hat{\mathbf{y}} + 0\,\hat{\mathbf{z}}\\ \amp = r\sin\theta\cos\phi\,\hat{\mathbf{x}} + r\sin\theta\sin\phi\,\hat{\mathbf{y}}\\ \amp = r\sin\theta\left(\cos\phi\,\hat{\mathbf{x}}+\sin\phi\,\hat{\mathbf{y}}\right). \end{align*}
      We note that
      \begin{equation*} \sin\theta\hat{\mathbf{r}}+\cos\theta\hat{\mathbf{\theta}} =\cos\phi\,\hat{\mathbf{x}}+\sin\phi\,\hat{\mathbf{y}} \end{equation*}
      and hence
      \begin{align*} \mathbf{A} \amp = r\sin\theta\left(\sin\theta\hat{\mathbf{r}}+\cos\theta\hat{\mathbf{\theta}}\right). \end{align*}
      We note that in both cylindrical and spherical polars, the vector field has no component in the azimuthal direction. This is because the original vector field is axisymmetric about the \(z\)-axis. Furthermore, in cylindrical polars, the vector field is purely radial, whereas in spherical polars it has both radial and polar components.
      🔗
      🔗
      🔗
    2. A vector field is given in Cartesian coordinates by \(\mathbf{A}=(y,-x,0)\text{.}\) Convert to cylindrical and spherical polar coordinates. What do you notice?
      Answer.
      For cylindricals, we have
      \begin{equation*} \hat{\mathbf{\theta}}=\cos\theta\,\hat{\mathbf{y}}-\sin\theta\,\hat{\mathbf{x}}. \end{equation*}
      Hence,
      \begin{align*} \mathbf{A} \amp = y\,\hat{\mathbf{x}} - x\,\hat{\mathbf{y}} + 0\,\hat{\mathbf{z}}\\ \amp = r\sin\theta\,\hat{\mathbf{x}}-r\cos\theta\,\hat{\mathbf{y}}\\ \amp=-r\hat{\mathbf{\theta}}. \end{align*}
      🔗
      For sphericals,
      \begin{align*} \mathbf{A} \amp = y\,\hat{\mathbf{x}} -x\,\hat{\mathbf{y}} + 0\,\hat{\mathbf{z}}\\ \amp = r\sin\theta\sin\phi\,\hat{\mathbf{x}} - r\sin\theta\cos\phi\,\hat{\mathbf{y}}\\ \amp = r\sin\theta\left(\sin\phi\,\hat{\mathbf{x}}-\cos\phi\,\hat{\mathbf{y}}\right). \end{align*}
      We note that
      \begin{equation*} \hat{mathbf{\phi}}=-\sin\phi\,\hat{\mathbf{x}}+\cos\phi\,\hat{\mathbf{y}}, \end{equation*}
      and hence
      \begin{equation*} \mathbf{A}=-r\sin\theta\hat{mathbf{\phi}}. \end{equation*}
      We note that in both cylindrical and spherical polars, the vector field is purely azimuthal.
      🔗
      🔗
      🔗
    3. A vector field is given in Cartesian coordinates by \(\mathbf{A}=(x,y,z)\text{.}\) Convert to cylindrical and spherical polar coordinates. What do you notice?
      Answer.
      For cylindrical polar coordinates, and using the answer to the first part of this question, we may immediately write
      \begin{align*} \mathbf{A} \amp = x\,\hat{\mathbf{x}} + y\,\hat{\mathbf{y}} + z\,\hat{\mathbf{z}}\\ \amp=r\hat{\mathbf{r}}+z\hat{\mathbf{z}}. \end{align*}
      For spherical polar coordinates, and using the answer to the first part of this question,
      \begin{align*} \mathbf{A} \amp = x\,\hat{\mathbf{x}} + y\,\hat{\mathbf{y}} + z\,\hat{\mathbf{z}}\\ \amp=r\sin\theta\cos\phi\hat{\mathbf{x}}+r\sin\theta\sin\phi\hat{\mathbf{y}}+r\cos\theta\hat{\mathbf{z}}\\ \amp=r\hat{\mathbf{r}}. \end{align*}
      The velocity field is purely radial in spherical polars, but has both radial and axial components in cylindrical polars.
      🔗
      🔗
      🔗
    4. A vector field is given in Cartesian coordinates by \(\mathbf{A}=(y,2x,z)\text{.}\) Write down the components of this vector field in cylindrical polars and in spherical polars.
      🔗
      Answer.
      In cylindrical polars, the unit vectors are related by
      \begin{equation*} \hat{\mathbf{x}}=\cos\theta\,\hat{\mathbf{r}}-\sin\theta\,\hat{\mathbf{\theta}},\quad \hat{\boldsymbol{y}}=\sin\theta\,\hat{\mathbf{r}}+\cos\theta\,\hat{\mathbf{\theta}},\quad \hat{\mathbf{z}}=\hat{\mathbf{z}}. \end{equation*}
      Hence,
      \begin{align*} \mathbf{A} \amp = y\,\hat{\mathbf{x}} + 2x\,\hat{\mathbf{y}} + z\,\hat{\mathbf{z}}\\ \amp = r\sin\theta(\cos\theta\,\hat{\mathbf{r}}-\sin\theta\,\hat{\boldsymbol{\theta}}) + 2r\cos\theta(\sin\theta\,\hat{\mathbf{r}}+\cos\theta\,\hat{\boldsymbol{\theta}}) + z\,\hat{\mathbf{z}}\\ \amp = r\sin(3\theta)\,\hat{\mathbf{r}} + r\cos(3\theta)\,\hat{\boldsymbol{\theta}} + z\,\hat{\mathbf{z}}. \end{align*}
      🔗
      In spherical polars, the unit vectors are related by
      \begin{equation*} \hat{\mathbf{x}}=\sin\theta\cos\phi\,\hat{\mathbf{r}}+\cos\theta\cos\phi\,\hat{\mathbf{\theta}}-\sin\phi\,\hat{\mathbf{\phi}}, \end{equation*}
      \begin{equation*} \hat{\mathbf{y}}=\sin\theta\sin\phi\,\hat{\mathbf{r}}+\cos\theta\sin\phi\,\hat{\mathbf{\theta}}+\cos\phi\,\hat{\mathbf{\phi}}, \end{equation*}
      \begin{equation*} \hat{\mathbf{z}}=\cos\theta\,\hat{\mathbf{r}}-\sin\theta\,\hat{\mathbf{\theta}}. \end{equation*}
      Hence,
      \begin{align*} \mathbf{A} \amp = y\,\hat{\mathbf{x}} + 2x\,\hat{\mathbf{y}} + z\,\hat{\mathbf{z}}\\ \amp = r\sin\theta\sin\phi(\sin\theta\cos\phi\,\hat{\mathbf{r}}+\cos\theta\cos\phi\,\hat{\boldsymbol{\theta}} - \sin\phi\,\hat{\boldsymbol{\phi}}) + 2r\sin\theta\cos\phi(\sin\theta\sin\phi\,\hat{\mathbf{r}}+\cos\theta\sin\phi\,\hat{\boldsymbol{\theta}} + \cos\phi\,\hat{\boldsymbol{\phi}}) + r\cos\theta(\cos\theta\,\hat{\mathbf{r}}-\sin\theta\,\hat{\boldsymbol{\theta}})\\ \amp = r\left(3\sin^2\theta\sin\phi\cos\phi+\cos^2\theta\right)\,\hat{\mathbf{r}} + r\sin\theta\cos\theta\left(3\sin\phi\cos\phi-1\right)\,\hat{\boldsymbol{\theta}} + r\sin\theta\left(2\cos^2\phi-\sin^2\phi\right)\,\hat{\boldsymbol{\phi}}. \end{align*}
